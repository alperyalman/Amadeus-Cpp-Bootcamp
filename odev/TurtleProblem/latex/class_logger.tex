\doxysubsection{Detailed Description}
Generete \doxylink{class_logger}{Logger} object to create simulation log file. 

\doxylink{class_logger}{Logger} class is used to create log file. All the updated information related to turtles and waterjets are logged into the specified file during the simulation.
