\doxysubsection{Detailed Description}
Calculate and check turtle and waterjet interactions. 

\doxylink{class_object_interaction}{Object\+Interaction} class is used to calculate turtle distance to waterjet, to calculate turtle angle with respect to x-\/axis, to check if the turtle is in the environment, and to check if the turtle is in wet area.
